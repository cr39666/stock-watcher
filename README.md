# 📈 Stock Watcher

一款基于 **Electron + Vue 3 + TypeScript** 的 A 股桌面盯盘工具，支持悬浮球模式，可常驻桌面实时监控自选股行情与持仓盈亏。

> Designed by **Croyell** 🌙

---

## ✨ 功能特性

### 🫧 悬浮球模式

- 随时可收缩为 **80×80** 透明悬浮球形式常驻桌面顶层
- 支持鼠标 **拖拽移动**，不占用任务栏
- 单击悬浮球展开至详情页，可随时在悬浮球 ⇄ 详情页间切换
- 全局快捷键（默认 `Alt+Z`）一键闪速显示/隐藏主界面或切换回悬浮球状态

### 📊 实时行情

- 接入 **腾讯财经行情接口**，支持 A 股（上海 / 深圳 / 北交所）全市场
- 行情数据 **每秒自动刷新**，包含最新价、昨收价、涨跌额、涨跌幅
- 输入 **6 位纯数字代码** 即可自动识别并补齐市场前缀（`sh` / `sz` / `bj`）

### 💼 持仓管理

- **添加自选**：输入股票代码，通过独立对话框 (`Modal`) 设置初始成本价和持仓手数
- **调仓操作**：支持加仓 / 减仓，加仓时自动计算 **加权平均成本**
- **删除 / 清空**：支持勾选多条后批量删除，或一键安全清空全部自选
- 数据基于 **localStorage** 本地持久化存储

### 📋 详情展示

- 表格展示核心数据列：名称、现价、当日盈亏 (D.PnL)、持仓总盈亏 (T.PnL)、成本均价/持仓市值、涨跌幅、持仓手数
- 涨跌以 **红涨蓝跌** 颜色直观标注
- **多档数据显示** 模式：
  - 点击表头名字列，在【仅名称】→【仅代码】→【名称+代码】三档循环切换
  - 点击成本列，在【均摊保本价】与【当前持仓市值】两档循环切换
  - 点击 Qty 列，在【持仓手数】与【价格提醒】两档循环切换
- **多列智能排序**：支持点击 Price / D.PnL / T.PnL / Chg% 表头切换升序、降序以及取消排序（恢复默认的拼音首字母排序）
- 底部汇总即时展示 **每日总盈亏** 和 **持仓总盈亏**

### 🔔 价格提醒

- 切换 Qty 列至 **价格提醒模式** 后，点击单元格可设置目标价格提醒
- 支持 **涨到** 和 **跌到** 两个独立方向，每只股票最多同时设置两个提醒
- 价格验证：涨到价格不低于当前价，跌到价格不高于当前价
- 打开设置弹窗时自动回显已设置的提醒信息（涨优先）
- 提醒触发时：
  - **页面内橙色Toast提示**，需用户点击确认关闭，支持多提醒排队显示
  - **系统通知弹窗** 同步推送
- 一键清除：点击目标价格旁的删除按钮可独立清除涨/跌提醒

### ⚙️ 系统设置 & 关于

- **个性化设置**：自带配置面板页，能独立启用/禁用"悬浮球"及"主窗口"的是否置顶 (Always on Top) 状态
- **热键录制**：在设置页面内可可视化定义并录制呼出主界面的全局快捷键
- **多语言支持**：内置中文、英文、默认（中英混合）三种语言模式，可在设置页面切换
- **关于界面 & 自动更新**：精致呈现 Logo 动画；支持**点击版本号按钮一键检测并平滑升级**至 GitHub 最新版本

### 🔒 隐私模式

- 一键点击底部 🔓 标志切换 **隐私遮盖**
- 所有敏感数据（名称、价格、盈亏、成本、手数）将自动被 `❇❇` 替代显示
- 适用于办公场景，防止敏感持仓信息外溢

### 🖱️ 交互细节

- 点击价格列自动 **一键复制** 到系统剪贴板
- 鼠标点击单行可高亮选中，再次点击取消
- 操作反馈通过内置的高复用 **Toast 提示**（成功 / 信息 / 警告 / 失败）进行友好播报
- 包括清空在内的高危操作均配有弹窗 **Confirm 二次确认拦截**，防止误点击操作丢失记录

### 🪟 无边框沉浸窗口

- 完全 **无边框透明** 窗口，圆角暗色主题，极致沉浸
- 集成 **iOS 风格拖拽条**，基于 IPC 手动拖拽计算偏移，稳定可靠避开 `-webkit-app-region` 局限
- 窗口大小随内容区配合 `ResizeObserver` 动态与主进程同步 **自适应调整**，阴影与边距适配无死角

## 🚀 快速开始

### 环境要求

- [Node.js](https://nodejs.org/) >= 18
- npm >= 9

### 安装依赖

```bash
npm install
```

### 开发调试

```bash
npm run dev
```

> 启动后将自动打开 Electron 窗口，支持 HMR 热更新。按 `F12` 可在任意窗口呼出控制台面板调试。

### 生产构建与发布

本项目的打包命令已默认集成 `--publish always`。如果需要打包后自动将安装包推送到 GitHub Release 以供用户在线更新，请先在当前环境（或 CI/CD 中）配置好具有 `repo` 权限的变量 `GH_TOKEN`：

```bash
# Windows
npm run build:win

# macOS
npm run build:mac

# Linux
npm run build:linux
```

## 📌 基础使用流向

1. **运行**：启动应用后默认展示主界面列表，随时可通过点击左下角 Logo 缩小为悬浮球，全局通过 `Alt+Z` 可闪速显示/隐藏主窗体
2. **入表**：在底栏输入目标 `6 位代码`，敲击回车或点击 `➕`，按弹窗输入建仓单价及总数量添加
3. **查阅**：表格顶端表头皆可供点击以完成不同状态切换和特定数值正/倒序排版
4. **调仓**：点击表中 `Qty(手数)` 列快速加仓或套现平仓
5. **提醒**：点击 Qty 表头切换至价格提醒模式，设置目标价格，系统将在股价触及目标时推送页面和系统双重提醒
6. **设置与关于**：点击底部齿轮可配置语言、全局显示热键或置顶规则；点击 `i` 图标进入关于界面，可点击底部版本号胶囊按钮检查更新
